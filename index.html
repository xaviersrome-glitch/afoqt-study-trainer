🎯 Test Strategy Quick Guide

Word Problems (Math): Read twice → underline what's being asked → write out knowns → choose formula → solve → check if answer makes sense.
Analogies: Always make a bridge sentence first. "A ___ is used to ___." Then test all 4 answer choices with the SAME sentence.
Vocabulary you don't know: Check the prefix/suffix/root → think of a word that sounds similar → use context (positive/negative tone) → eliminate obviously wrong answers.
Reading Comprehension: Read the questions FIRST so you know what to look for. The answer is ALWAYS in the passage — don't use outside knowledge.
Time management: Don't spend more than 60–90 seconds per question. Mark it and move on — return if time allows.
