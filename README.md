# ✈️ AFOQT Study Trainer

An AI-powered Math & Verbal practice app for the AFOQT exam — endless questions, full explanations, and real study tips.

## Features
- 📐 **Math mode** — Arithmetic, algebra, word problems, geometry
- 📖 **Verbal mode** — Vocabulary, analogies, word knowledge
- ⚡ **Mixed mode** — Random math + verbal
- 3 difficulty levels: Easy → Medium → Hard
- AI-generated questions so you never run out
- Full explanations after every answer
- Hints, streak tracking, and AFOQT-specific study tips

## How to Use
Just open `index.html` in your browser — no install, no setup needed.
